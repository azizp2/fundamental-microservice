# MICROSERVICE LEARNING ROADMAP SUMMARY

## Goal

Membangun mini e-commerce berbasis Microservice menggunakan:

* .NET
* CQRS + MediatR
* RabbitMQ
* Saga Pattern
* Outbox Pattern
* Retry Queue
* Delayed Message

Fokus utama:

* Memahami konsep
* Bukan sekadar membuat fitur
* Semua checkpoint harus direview sebelum dianggap selesai
